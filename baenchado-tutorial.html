วิธีการเขียนสีน้ำ ภาพทิวน์ทัศน์แบนชะโด

By บุ๊ค ชัยนุวัตร

 แบ่งปันขั้นตอนการเขียนสีน้ำภาพทิวทัศน์ครับ ซึ่งภาพที่ผมจะสาธิตในวันนี้เป็นภาพถนนแบนชะโด ซึ่งอยู่ทางชานเมืองกรุงเทพนี้เองครับ

เรามาดูขั้นตอนกันครับ

Drawing

ในการร่างภาพ ผมมักจะวางเส้นขอบฟ้า (Horizontal Line) ค่อนไปทางด้านบน หรือ ไม่ก็ด้านล่าง และใช้ดินสีเนื้ออ่อนในการร่างภาพครับ

ปกติผมชอบใช้  ดินสอ 2B ครับ ไม่เข้มและอ่อนจนเกินไป กำลังดีครับ

Layer 1

เริ่มต้นด้วยการลงสีชั้นแรก ภาพนี้ผมจะเริ่มจากการลงสีบรรยากาศโดยรวม และเป็นค่าน้ำหนักอ่อนสุดก่อนครับ 

ท้องฟ้าและถนนด้านล่าง ผมใช้สี yellow ochre และ ultramarine ลงสีแบบเปียกบนแห้ง (Wet on Dry) โดยใช้วิธีระบายแบบระบายเรียบหลายสี

 Layer 2 

หลังจากทิ้งไว้จนแห้งสนิท(อาจใช้ไดรย์เป่าผมช่วยให้แห้งไวขึ้นได้ครับ)

ผมเริ่มลงสีแนวต้นไม้ในระยะไกล ผมใช้สี Sap green + Viridian + Ultramarine + แตะ Payne grey นิดหน่อย ระบายแบบเปียกบนแห้งเหมือนเดิมครับ

ในขณะที่สียังเปียกอยู่ ผมใช้สี Sap green + Viridian + Prussian blue แบบข้น ระบายส่วนด้านล่างของแนวต้นไม้

และเริ่มระบายต้นไม้ทางด้านขวาด้วยสีเขียวหลายๆเฉด เพื่อให้ภาพดูไม่น่าเบื่อครับ

N การที่เราผสมสีโทนน้ำเงิน หรือ โทนเทา เข้าไปด้วยช่วยผลักระยะให้วัตถุนั้นๆดูไกลออกไปครับ

Layer 3 

ระบายต้นไม้ระยะหน้าทางด้านซ้ายด้วย Yellow ochre + Sap green + Viridian และมีการแตะสีแดง Alizarine crimson + Payne grey นิดหน่อยเพื่อเบรกสีและทำเป็นส่วนของเงาต้นไม้ครับ

ในขั้นตอนนี้ จะเห็นได้ว่าผมใช้เทคนิคการเช็ดสีออกโดยใช้พูกันจุ่มน้ำซับหมาดๆ เช็ดสีออก เพื่อทำลายรูปทรง ให้ภาพดูไม่เป็นแผ่นเรียบๆ เพิ่มความน่าสนใจ และบรรยากาศของภาพครับ 

N ในภาพที่มีการซ้ำกันของสีมากๆ เราอาจใช้วิธีการเพิ่มสีอื่นๆ เพิ่มบ้าง ลดบ้าง เพื่อให้เกิดความหลากหลายของสีมากขึ้น

 

Layer 4

เริ่มลงสีส่วนของตัวลำต้นใช้สี Burnt sienna + Prussian blue + แตะ Payne grey นิดๆ เพื่อเบรกสีเป็นส่วนของเงา ระบายลงมาเชื่อมต่กับเงาบนถนน

ส่วนของเขาบนถนนใช้สี Ultramarin + Purple + Alizarine crimson

Layer 5 

ระบายส่วนของคนและรถยนต์

ส่วนของคนในระยะหน้าอยู่ในเงา ผมระบายขา ให้กลืนไปกับเงาบนพื้นถนน

และตกแต่งรายละเอียด เพิ่มเสาไฟฟ้า ด้วยเทคนิคแห้งบนแห้ง และใช้สีขาวแต้มบางจุดครับ

ใครมีข้อสงสัย หรือ ต้องการพูดคุยแลกเปลี่ยนความรู้ทางศิลปะ สามารถอินบ๊อกซ์มาหาผมได้เลยครับ ผมยินดี
